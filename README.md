Pokémonia

https://github.com/supermario7777/andersen-dev-proj
Description: A responsive web application that interacts with the PokéAPI to browse and
compare Pokémon.
Tech Stack
React 18
TypeScript
Vite
Axios
CSS3 (Responsive Layouts)
Key Features
Dynamic Data Fetching (1,000+ items)
Favorites System (Localized state management)
Comparison Tool for attribute analysis
Mobile-first Responsive UI
How to Run
1. Clone the repository.
2. Run npm install .
3. Start the project with npm run dev or npm start .
